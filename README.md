# CS5053-Graphics-Project
Treasure Collector 3D Game for Computer Graphics

## Installation
### Clone the repo
```bash
git clone https://github.com/Sadita06/CS5053-Graphics-Project.git
```

## Description
A 3D WebGL game where the player explores a world and collects treasures.

## Controls
- WASD -> Move
- E -> Collect items

## Features
- WebGL rendering pipeline
- Phong lighting (ambient, diffuse, specular)
- Textured objects
- Collision detection
- Interactive gameplay
